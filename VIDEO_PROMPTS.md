```tsx
// Background video component for sections
<div className="absolute inset-0 overflow-hidden">
  <video
    autoPlay
    muted
    loop
    playsInline
    className="w-full h-full object-cover opacity-20"
    poster="/images/{matching-hero-image}.jpg"
  >
    <source src="/videos/{clip-name}.mp4" type="video/mp4" />
  </video>
  <div className="absolute inset-0 bg-gradient-to-b from-background/60 via-background/80 to-background" />
</div>
```

### Performance considerations:
- Serve as compressed MP4 (H.264, ~2-4MB per 5s clip)
- Use `poster` attribute with existing hero images for instant display
- `opacity-20` keeps videos subtle as backgrounds
- Gradient overlay ensures text readability
- Consider `prefers-reduced-motion` media query to disable for accessibility
- Lazy-load off-screen videos with IntersectionObserver
